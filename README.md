# CodeAlpha FAQ Chatbot

## Description
The CodeAlpha FAQ Chatbot is an AI-powered desktop application developed as part of the CodeAlpha AI Internship.  
It uses NLP techniques such as TF-IDF Vectorization and Cosine Similarity to answer frequently asked questions.

The chatbot provides a simple GUI using Tkinter for user interaction.

---

## Features
- Answers frequently asked questions  
- NLP-based response system (TF-IDF + Cosine Similarity)  
- Interactive GUI using Tkinter  
- Handles multiple user queries  
- Fast and lightweight chatbot  

---

## Technologies Used
- Python  
- Tkinter (GUI)  
- Scikit-learn
